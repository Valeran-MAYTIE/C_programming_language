### Statements and Blocks

    {
        /* block */
        printf("hello");

### If-Else

```c
    if (expression)
        statement1
    else
        statement2
```

### Else-If

```c
    if (expression)
        statement
    else if(expression)
        statement
    else if(expression)
        statement
    else
        statement
```
